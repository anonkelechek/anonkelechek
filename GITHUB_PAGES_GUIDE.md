# 🚀 Гайд по публикации КЕЛЕЧЕК АНОН на GitHub Pages

Этот файл всегда будет лежать у тебя в папке проекта как шпаргалка. Если ты захочешь обновить сайт (например, добавить новые функции), просто следуй этим шагам.

---

## 🛠 Подготовка (Делается только один раз)

1. Открой терминал в VS Code (Вкладка Terminal -> New Terminal).
2. Выполни команду для установки инструмента публикации:
   ```bash
   npm install gh-pages --save-dev
   ```
3. Зайди на [GitHub.com](https://github.com/), нажми **New repository**.
4. Назови его, например, `kelechek-anon` (без пробелов).
5. Не ставь галочку "Add a README", просто нажми **Create repository**.
6. GitHub выдаст тебе ссылку вида `https://github.com/Laziza/kelechek-anon.git`. Скопируй её.

---

## 📤 Публикация сайта (Шаги для первого деплоя)

В терминале твоего редактора по очереди введи эти команды (замени `<ТВОЯ_ССЫЛКА>` на скопированную ссылку из шага 6):

```bash
git add .
git commit -m "Первый запуск КЕЛЕЧЕК АНОН"
git branch -M main
git remote add origin <ТВОЯ_ССЫЛКА>
git push -u origin main
```

Теперь код загружен на GitHub. Чтобы выложить сам сайт в интернет, введи:
```bash
npm run deploy
```

🎉 **Готово!** Через 2-3 минуты твой сайт появится по ссылке: 
`https://<ТВОЙ_НИК>.github.io/<ИМЯ_РЕПОЗИТОРИЯ>/`

---

## 🔄 Как обновлять сайт в будущем?

Если ты поменяла код (например, изменила цвет или добавила кнопку) и хочешь обновить сайт в интернете, тебе нужно выполнить всего 3 команды в терминале:

1. Сохранить изменения в Git:
   ```bash
   git add .
   git commit -m "Обновила дизайн"
   ```

2. Отправить изменения на GitHub:
   ```bash
   git push
   ```

3. Обновить сайт в интернете:
   ```bash
   npm run deploy
   ```
